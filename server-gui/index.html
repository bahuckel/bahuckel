Owner account & Giphy

Creates the first admin user to manage the server and view password reset requests. A Giphy API key is required for the in-app GIF picker (free tier at developers.giphy.com).

Open in browser

Run bahuckel-website.exe (or start-website.bat in the release folder) for the marketing site on 8080. The chat app is on 3001 when the server is running — separate processes so you can restart chat without stopping the site.

Cloudflare Tunnel

Exposes your local server to the internet via a temporary URL. Others can join without port forwarding. Requires Cloudflared installed (cloudflare.com/docs).
